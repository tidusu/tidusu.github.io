上传到git时发生 fatal: unable to access ‘https://github.com/xxx': The requested URL returned error: 403

解决方法

  1. 在 repo 目录中 编辑 .git/config 文件
  2. 在[remote "origin"]后找到 url=entry
  3. 修改url=https://MichaelDrogalis@github.com/derekerdmann/lunch_call.git 为 url=ssh://git@github.com/derekerdmann/lunch_call.git
  4. 保存并退出
