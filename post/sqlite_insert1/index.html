如何在sqlite中实现UPSERT UPDATE/INSERT 效果

October 06, 2019
数据库

有时候我们希望在sqlite中插入新数据，又不想用到REPLACE INTO，因为它类似于DELETE和INSERT的效果，如果想达到UPSERT UPDATE/INSERT的效果，那应该怎样做呢。

假设表格user有三列数据，id，name，info，已知待插入数据的id为1的、name为Bob，正确写法如下：

INSERT OR REPLACE INTO user (id, name, info) 
  VALUES (  1, 
            'Bob',
            COALESCE((SELECT info FROM user WHERE id = 1), 'no info')
          );
  • 当user表中存在id为1的数据时，它的name将被替换为Bob，原先的info保持不变;
  • 当数据中不存在id为1的数据时，将插入一条新的数据，COALESCE的用法是返回第一个非空结果，全为空是则返回空值，因此新记录的info将使用默认的no info
